# Goodcheck Rules

> A collection of useful [Goodcheck](https://github.com/sider/goodcheck) rules.

## Usage

Add a URL to `import:` on your `goodcheck.yml`. For example:

```yaml
rules:
  # ...

import:
  - https://raw.githubusercontent.com/sider/goodcheck-rules/master/rules/typo.yml
  - # others...
```
